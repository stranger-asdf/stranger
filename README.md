# � Happy Bday Website

This a simple birthday wish website i made for a friend. It has some react and tailwind code. Very responsive and looks good on phone too.

## 🚀 How to use

1. **Download**:
   ```bash
   git clone <your repo link here>
   cd BIRTHDAY-WISH
   npm install
   ```

2. **Run it**:
   ```bash
   npm run dev
   ```

## 🎨 How to change things

- Open `src/components/Hero.tsx` to change the main name.
- Put your photos in `public/images/`. Name them same like gallery1.jpeg, gallery2.jpeg etc.
- Change quotes and poems in `src/data/poems.json` and `src/data/quotes.json`.
- If u want to change background or colors, it is in `src/index.css` u can edit if you want.
- In `src/pages/Index.tsx` u can add or remove sections.

##  Project Structure

- `src/components`: all the UI parts
- `src/data`: json files to change text
- `public/images`: where pictures go

## 💻 Tech i used

- React
- Vite
- Tailwind CSS

Thats it, enjoy the birthday website!
